# スクレイピングルール

## Google Maps
- 無限スクロール方式。スクロールが止まったら取れた分で完了（部分成功）
- Stage回復は不要。リトライしても同じIPでブロックされるだけ

## 共通: 停止検知と早期回復
- 「セッション開始中」「ページ読み込み中」で長時間（15秒以上）止まったら異常
- 異常検知時は以下を素早くローテーション:
  1. プロファイルリセット（新UUID）
  2. フィンガープリント再生成（新セッション）
  3. Cookie クリア＆再取得
  4. IP変更（Tor回線更新）
- 90秒待ってからリトライではなく、**15-30秒で切り替え**
- ダメなものは早く諦めて次の手を打つ

## コスト最適化
- 設計・分析・判断はOpus
- 実装・コーディングはSonnet（サブエージェント）

## デプロイ
- デプロイ中もDiscordで返信可能にする（バックグラウンド実行、pollでブロックしない）
- デプロイ確認は必ず事前に取る
